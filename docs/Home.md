# Forge (Minimal Branch)

A stripped-down Rules Engine for Magic: The Gathering containing only the desktop GUI and AI simulation tools.

Forge is open source software released under the GNU Public License. Not affiliated with Wizards of the Coast.

## Included Modules

- **forge-core** - Core game engine and rules
- **forge-game** - Game session management
- **forge-gui** - UI components and card scripting resources
- **forge-gui-desktop** - Java Swing desktop client
- **forge-ai** - AI opponent logic and simulation
